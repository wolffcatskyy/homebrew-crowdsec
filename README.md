# homebrew-crowdsec

Homebrew tap for CrowdSec community tools.

## Install

```bash
brew tap wolffcatskyy/crowdsec
```

### CrowdSec Blocklist Import

Memory-efficient importer for 28+ public threat feeds into CrowdSec via LAPI.

```bash
brew install wolffcatskyy/crowdsec/crowdsec-blocklist-import
```

### CrowdSec UniFi Bouncer

CrowdSec bouncer for UniFi Dream Machine devices using native nftables.

```bash
brew install wolffcatskyy/crowdsec/crowdsec-unifi-bouncer
```
